## About This Project
This project provides examples of design patterns.

## Used Technologies
[Laravel 10](https://laravel.com/) with PHP 8.2.9. 

## Getting Started
1. Clone the repository

2. Install dependencies
```
componser install
```

3. Run the server
``` 
php artisan serve
```

## Run tests
``` 
php artisan test
```
